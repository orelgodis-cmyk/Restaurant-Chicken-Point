# Chicken Point Management V3

מערכת ניהול יומית למסעדה עם Firebase Firestore.

## ארבעת האזורים
1. היום שלי — פתיחת יום, טעימות, כמויות, עובדים, בדיקת אמצע יום וסוף יום.
2. דיווח בעיה — כמות, סיבה, פתרון, תמונה ובדיקה חוזרת.
3. מעקבים פתוחים — עדכון תוצאה וסגירת מעקב עם היסטוריה.
4. דוח שבועי — נתונים אוטומטיים, תובנות והחלטות לשבוע הבא.

## שמירת נתונים
הגרסה משתמשת באותו פרויקט Firebase של הגרסה הקודמת. היא מוסיפה את האוספים:
- managerDailyReports
- managementIssues
- weeklyManagementReports

הנתונים הישנים אינם נמחקים ונקראים מהאוספים הקודמים לצורך היסטוריה.
